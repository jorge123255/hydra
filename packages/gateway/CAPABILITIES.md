
## Security Advisor Mode

Use `/security` command or ask security-related questions to activate Security Advisor mode:

```
/security Explain the CIA triad
/security How would you mitigate a SQL injection vulnerability?
/security What are the key differences between symmetric and asymmetric encryption?
/security Analyze this vulnerability: CVE-2024-1234
/security Help me prepare for the CISSP exam - Security Architecture domain
```

Features:
- **CISSP Exam Preparation**: Get explanations of all 8 CISSP domains
- **Vulnerability Analysis**: Structured threat analysis with mitigation strategies
- **Security Frameworks**: References to NIST, ISO 27001, OWASP, MITRE ATT&CK
- **Compliance Guidance**: GDPR, HIPAA, PCI DSS requirements
- **Secure Coding**: OWASP Top 10 explanations and prevention techniques
- **Incident Response**: Step-by-step guidance for security incidents

The system automatically detects security questions and routes them to security-optimized AI models.
